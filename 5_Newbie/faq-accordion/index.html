FAQs

  • What is Frontend Mentor, and how will it help me?

    Frontend Mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML, CSS and JavaScript. It's suitable for all levels and ideal for portfolio building.

  • What is the difference between Frontend Mentor and other coding challenges?

    Frontend Mentor challenges are designed to test your skills in HTML, CSS and JavaScript. Other coding challenges can be more challenging and require you to learn a new framework or programming language.

  • Is Frontend Mentor free?

    Yes, Frontend Mentor offers both free and premium coding challenges. The free challenges are available for all levels and are suitable for anyone looking to improve their frontend skills.

  • Can I use Frontend Mentor projects in my portfolio?

    Yes, you can use Frontend Mentor projects in your portfolio. Just make sure to credit us and link to our website.

  • How can I get help if I'm stuck on a challenge?

    If you're stuck on a challenge, you can join our Discord community and ask for help from our friendly community members. You can also join our Slack community for more support.
