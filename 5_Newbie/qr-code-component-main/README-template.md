# Frontend Mentor - QR code component solution

This is a solution to the [QR code component challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/qr-code-component-iux_sIO_H).

## Overview
- Challenge: Create a QR code component that adjusts responsively across different screen sizes
- Goal: Match the provided Figma design while ensuring proper responsiveness

## Features
- Responsive QR code card that maintains design integrity across devices
- Mobile-first development approach
- Proper content spacing with flexbox layout
- Smooth scaling from mobile to desktop views

## Built with
- Semantic HTML5
- CSS custom properties
- Flexbox
- Mobile-first workflow
- Media queries for responsive design

## Key Learnings
- Converting fixed Figma heights (499px) to flexible web layouts using `min-height`
- Implementing responsive design with media queries
- Managing content overflow with proper padding and spacing
- Using Flexbox for centered layouts and content distribution
